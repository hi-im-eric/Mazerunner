# Enterprise Domain Theme — Asset Generation Prompts

Use these prompts with an image generator (Gemini, DALL-E, Midjourney, etc.) to create visual assets for each MazeRunner enterprise domain theme. Each domain gets a simplified asset set: 1 sidebar decoration, 1 content background, and 3 card decorations (5 assets per domain, 40 total across 8 domains).

## Global Style Guide (include with every prompt)

All assets across all domains should share this visual language:

- **Style:** Clean, modern corporate illustration — realistic but slightly stylized
- **Palette:** Muted, professional tones appropriate to each domain. Avoid oversaturation.
- **Lighting:** Soft, even studio-style lighting with subtle depth. No dramatic shadows.
- **Texture:** Clean surfaces, subtle material detail (brushed metal, matte plastic, crisp paper)
- **Background:** Solid white or very light neutral gray (#F5F5F5) for easy compositing over any UI color scheme
- **Render style:** High-quality 3D render or flat-shaded vector illustration, suitable as UI decoration at low opacity (20–35%)
- **Composition:** Subject centered with generous negative space around edges. No text, watermarks, or UI elements in the image.

# Production Notes

- **All card assets must be 800x800 px** so they render at consistent visual weight across the UI
- **Content backgrounds must be 1600x900 px** (16:9) to cover widescreen content areas
- **Sidebar decorations must be 800x800 px** (square) to fit the sidebar panel
- After generation, verify that each asset works at 20–35% opacity over both light and dark backgrounds
- White/light gray backgrounds composite cleanly over any UI color scheme — unlike the dark fantasy assets, these should NOT use black backgrounds
- If the generator adds watermarks, they will need to be removed before embedding
- Assets will be resized and base64-encoded for embedding in the standalone HTML file
- Generate all assets for one domain at a time to maintain visual consistency within that domain
- Total: 40 assets (5 per domain × 8 domains)
